A modern web application featuring an interactive 3D ASCII cube, built with [Next.js](https://nextjs.org/), [Three.js](https://threejs.org/), and [shadcn/ui](https://ui.shadcn.com/). An open-source implementation of the ASCII Cube on [GitHub Shop](https://thegithubshop.com/).

## Features

- **3D ASCII Rendering**: Renders a 3D cube using ASCII characters via Three.js `AsciiEffect`.
- **Interactive Controls**:
  - **Rotate**: Drag with your mouse or finger, or use keyboard arrow keys.
  - **Zoom**: Use the scroll wheel to zoom in and out.
  - **Auto-rotate**: Toggle auto-rotation with the Play/Pause button.
- **Modern UI**: Styled with Tailwind CSS and accessible components from shadcn/ui.
- **Responsive Design**: Adapts to different screen sizes.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Library**: [Three.js](https://threejs.org/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

- **Clone the repository:**

    ```bash
    git clone https://github.com/ZhanZiyuan/ascii-cube.git
    cd ascii-cube
    ```

- **Install dependencies:**

    ```bash
    npm install
    ```

- **Run the development server:**

    ```bash
    npm run dev
    ```

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
